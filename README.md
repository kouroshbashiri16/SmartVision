## ✨ Features

### 🎯 Core Features
- **Real-time Detection**: Instant person detection using YOLOv8
- **Live Camera Feed**: Works with any USB or built-in webcam
- **Dark Modern UI**: Eye-friendly dark theme with neon accents
- **Performance Optimized**: Runs on CPU with 30+ FPS
- **Adjustable Parameters**: Fine-tune confidence and IOU thresholds

### 📊 Live Statistics
- **FPS Counter**: Real-time performance monitoring
- **Person Count**: Live count of detected people
- **Detection Confidence**: Visual confidence indicator

### 🎮 Interactive Controls
- **Start/Stop**: Control detection on-demand
- **Confidence Slider**: Adjust detection sensitivity
- **IOU Slider**: Control overlap between detections
